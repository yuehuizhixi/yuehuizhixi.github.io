视图

视图中的数据是根据预定义的选择条件从一个或多个行集中生成的。用视图可以定义一个或多个表的行列组合。

视图的内容由查询定义，而视图中的数据并不像表、索引那样需要占用存储空间，视图中保存的仅仅是一条select语句，其数据来自于视图所引用的数据库表或者其他视图，对视图的操作与对表的操作一样，可以对其进行查询、修改、删除。

优点：

（1）保护数据安全

（2）简化操作

（3）使分散数据集中

（4）提高数据的逻辑独立性

创建视图：

1、

create view student_view1 as select * from student;

2、

create view student_view2（sname,cname,grade）
as select sname,cname,grade
from student ,course,sc
where student.sno=sc.sno and course.cno=sc.cno;

3、

create view student_birthyear（sno,sname,birthyear）
as 
select sno,sname,2010-sage
from student

4、

create view student_avg（sno,avggrade）
as 
select sno,avg（grade）from sc
group by sno;

查看视图

describe view_name；
dec view_name
show table status like ‘view_name’
show create view ‘view_name’
select * from information_schema.views where table_name=’ view_name’

3.3.6 确定有限自动机的化简
对一个DFA M最少化的基本思想:
把M的状态集划分为一些不相交的子集，使得任何两个不同子集的状态是可区别的，而同一子集的任何两个状态是等价的。最后，让每个子集选出一个代表，同时消去其他状态。

二阶线性微分方程解的结构

y" + p(x)y' + q(x)y = f(x)
对于齐次方程  y" + p(x)y' + q(x)y = 0  的通解：
求出两个线性无关的特解，然后y(x) = C1y1(x)+C2y2(x)

设y1(x)、y2(x)是非齐次线性方程y" + p(x)y' + q(x)y = f(x)两个不同的特解，
则y1(x)-y2(x)是对应齐次方程y" + p(x)y' + q(x)y = 0的一个特解。

因此y" + p(x)y' + q(x)y = f(x)的通解为，特解+对应其次的通解：
y = y*(x) + C1y1(x)+C2y2(x)
